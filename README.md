#  Personal Finance Tracker

A simple web application to manage personal finances, built with Flask.

## 🚀 Features

- Add income and expenses
- View transaction history
- Filter by type (income/expense)
- Automatic balance calculation

## 🛠️ Tech Stack

- Python
- Flask
- HTML/CSS
- JSON (data storage)
